Why Interpreter?

  1. Reduced memory usage Compiled to a concise bytecode, rather than machine code
  2. Reduced parsing overhead Bytecode is concise, allowing eager compilation of JS source
  3. Reduced compiler pipeline complexity Bytecode is source-of-truth for optimizing/deoptimizing
